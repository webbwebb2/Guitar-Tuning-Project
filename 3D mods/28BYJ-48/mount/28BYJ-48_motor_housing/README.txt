# Summary

Housing for the ubiquitous 28BYJ-48 stepper motor that can be screwed to a flat surface.

# Print Settings

Printer: Wanhao 4S
Rafts: No
Supports: Doesn't Matter
Resolution: 0.2
Infill: 20%

Notes: 
Didn't need support on my printer using ABS and active cooling. This was intended to be a close fit on the motor and it is so using my printer, if dimensions turn out different on yours then you can easily scale the whole thing up/down a few percent if needed.
